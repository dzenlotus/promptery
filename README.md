# Promptery

Kanban-based context management for AI agents via MCP.

Reusable prompt primitives — roles, skills, rules, tools — composed into live tasks on a kanban board. Your AI agent gets full context through MCP: role, rules, skills, task description, related tasks.

Run a subagent with one profile to optimize React performance. Run another with a different profile to audit accessibility. Same CLI slot, different competencies from the board.

One terminal command, opens in browser, agent connects automatically.
