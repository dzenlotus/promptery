## Guidelines

- Use 2x / Retina captures for sharpness on high-DPI displays.
- Crop to the relevant area; avoid exposing real personal data or project names.
- Compress with a lossless PNG optimizer (e.g. `pngquant`, `oxipng`) before committing.
- Keep filenames lowercase with hyphens; match the names in the table above to avoid broken README links.
